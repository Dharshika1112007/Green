🌿 Greenden – Plant Store Website
Greenden is a simple responsive plant store website built using HTML and Tailwind CSS (CDN version).
It showcases a landing page with a navbar, hero section, and an about section with feature cards.
🚀 Features
✅ Responsive Navigation Bar
✅ Hero Section with Call-to-Action buttons
✅ About Section with 3 Feature Cards
✅ SVG Icons for visual design
✅ Mobile Responsive Layout using Tailwind Flexbox
✅ Sticky Navbar
🛠️ Technologies Used
HTML5
Tailwind CSS (CDN)
SVG Icons (Heroicons style)
